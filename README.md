# artifacts

Self-contained HTML artifacts generated by Claude Code, published via the
[`publish-html`](https://github.com/alilloig/dotclaude/tree/main/skills/publish-html)
skill for cross-repo sharing.

## URL pattern

`https://alilloig.github.io/artifacts/<slug>/`

## Hosting

Each artifact lives at `docs/<slug>/index.html`. GitHub Pages serves the
`/docs` folder from `main`.

## Privacy policy

Anything in this repo is **public**. Sensitive artifacts (audits, PR reviews,
security findings, client work) MUST NOT land here — they go to secret gists
instead. The `publish-html` skill enforces a mandatory `AskUserQuestion`
sensitivity gate to prevent accidents.

## Retention

Artifacts older than 90 days may be pruned. To unpublish, delete the slug
directory and push.
